# Assignment 1 - Reactive Shiny App (Iris Dataset)
This project implements a reactive web application using R Shiny 
to explore the Iris dataset.

## Features
- Reactive scatter plot with selectable X and Y variables
- Optional regression line
- Dynamic histogram with adjustable bins
- Custom summary statistics by selected variable
- Interactive data table with column selection

## Technologies Used
- R
- Shiny
- ggplot2
- dplyr
- bslib
